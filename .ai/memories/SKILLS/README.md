# ADDS 技能库

> 此目录存放 Agent 自进化的技能文档。
> 技能遵循 agentskills.io 开放标准，支持渐进式披露加载。

## 技能加载层级

| 层级 | 方法 | 加载内容 | Token 消耗 |
|------|------|----------|-----------|
| Level 0 | `skills_list()` | 名称 + 描述 + 类别 | ~50/skill |
| Level 1 | `skill_view(name)` | 完整内容和元数据 | ~200-500/skill |
| Level 2 | `skill_view(name, path)` | 特定参考文件 | ~500-2000/skill |
